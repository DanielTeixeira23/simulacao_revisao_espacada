# Simulação Revisão Espaçada

Este repositório contém o código utilizado no Trabalho de Conclusão de Curso (TCC) para a realização de simulações computacionais controladas de revisão espaçada, com foco na análise do desempenho do modelo **HLR (Half-Life Regression)** em diferentes horizontes de prática e parâmetros de espaçamento entre as revisões.
